CARA MEMBUAT BOT WHATSAPP DENGAN BAILEYS BY ADIWAJSHING.

  • MATCHBOX
  • 25 Januari 2023
  • 9 Comments

Halo Semuanya Kali ini saya akan berbagi tentang cara membuat bot whatsapp dengan menggunakan plugin baileys dari adiwajshing. Baileys Merupakan salah satu plugin yang menggunakan pengkodean python dan javascript yang dimana plugin ini menampilkan qr ke terminal command.

Langsung saja ke tutorial, pertama siapkan aplikasi pendukung seperti github dan google cloud.

Hal Yang paling penting pada project ini adalah plugin baileys karena plugin tersebut lah yang akan menghubungkan whatsapp ke server bot

Langkah pertama install plugin baileys dengan menggunakan tutorial Disini. Lalu gunakan kode "case" untuk membuat sebuah percakapan pada bot gunakan juga "button" untuk membuat button. Setelah Itu buat file config.js, untuk contohnya kunjungi Github Kami

Jika Kamu Masih bingung cara membuat bot nya bisa langsung ke github kami Disini, Jangan lupa mengganti file config.js nya yah. Jangan merekode script kami agar tidak ada yang error jika ingin merecode sc kami silahkan bisa langsung di fork yah, untuk tutorialnya bisa lihat di YouTube. Script bot whatsapp ini juga gratis, untuk apikey nya bisa langsung hubungi kami lewat whatsapp yah.

Jika Anda masih bingung juga cara membuat bot whatsapp ini anda bisa membeli nya di matchbox store caranya cukup whatsapp kami di Sini. Harga nya cukup terjangkau mulai dari 150K anda sudah bisa membelinya yang dalat aktif selama 8760 Jam, Atau Sekitar 365 Hari.
